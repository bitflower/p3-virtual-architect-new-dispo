# New Dispo Tools Plugin

Local plugin for managing the New Dispo project repositories.

## Skills

### `/update-repos`

Fetches and pulls master/main branches for all repositories in the Code folder.

**Usage:**
- Just say "update repos" or "pull all repos"
- Or use the command: `/update-repos`

**What it does:**
- Updates all four New Dispo component repositories
- Automatically detects master vs main branches
- Provides clear status output

## Installation

This plugin is automatically available when working in the New Dispo project directory.
