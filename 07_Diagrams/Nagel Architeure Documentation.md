# Nagel Architeture Documentation

## Architecture Design Diagrams

We are using `PlantUML` to describe architure as diagrams. The diagrams are stored as code. The ycan be rednered with a `PlantUML` rendering engine e.g. The VS Code plugin.

We are using the C4 model to document the different layers of the architecture.

We use the C4 plugin for `PlantUML` for that:
https://github.com/plantuml-stdlib/C4-PlantUML
