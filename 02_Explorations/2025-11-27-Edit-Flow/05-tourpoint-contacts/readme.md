# Tourpunkt Adressdaten ändern

- Wir ändern nur den `PERS` Satz zum Tourpunkt
- Auflösung findet über `RES_HST_ZUS` statt anhand eindeutigem Key `PERS_TIX` und `typ=999` (`pTourOrt_Lib`)
- Referenz: `V_TOUR`: Dort sieht man, wie Tourpunkt (=`res_hst`) mit `PERS` verknüfpt ist
- Der `PERS`-Datensatz für einen Tourpunkt ist immer vorhanden
